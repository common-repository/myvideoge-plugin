This Plugin allows you to put Myvideo.Ge videos easily in your Wordpress Blog Posts and Pages.

== Description ==

This Plugin will help you to put videos in your posts and pages from [Myvideo.Ge](http://myvideo.ge/) as easy as from [Youtube.com](http://youtube.com), [Vimeo.com](http://vimeo.com) and etc.

<strong>Features</strong><br />
* Change video alignment (left,center,right). default:left <br />
* Change video embed dimensions. default:452x380px


== Installation ==

Installation of Myvideo.Ge Plugin is very easy. You can use built-in Wordpress Plugin installer or:

1. Download Myvideo.Ge Plugin (.zip) and unzip it.
1. Upload the `myvideoge-plugin` folder on your FTP Server in folder `/wp-content/plugins/`.
1. Go to Wordpress Admin Panel --> Plugins and activate it.


== Frequently Asked Questions ==


= How can I use it? =

After you've installed Plugin, you have to put full URL of video from Myvideo.Ge in your Blog Post or Page and it will appear as embed.

Example:
`http://www.myvideo.ge/?video_id=760270`

= What if I just want to put URL and not let it appear as embed? =

If you want to put just URL of video and want to let it stay as it is, you should write some character before 'http://' protocol or just simply remove it.

Examples:
`_http://www.myvideo.ge/?video_id=760270` <br />
`www.myvideo.ge/?video_id=760270`
